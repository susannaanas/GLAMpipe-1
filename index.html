Metapipe

2*node-based metadata processing tool

View project on GitHub

The main purpose of Metapipe is to provide method for (meta)data manipulation and conversion. A typical use case would be importing free images from Flickr to Wikimedia Commons, for example.

Node-based GUI

Metapipe uses nodes for graphical user interface. With nodes it is possible to make easy to follow structure for all editing that is needed for a dataset.

koe

Technology

Metapipe is a node.js application and it uses MongoDB for data storage. It can be run locally (see installation) or it can be used as a web service.
